Pengukur Jarak untuk Hunting Burung

Aktifkan kamera, arahkan ke burung, klik untuk mengetahui jarak secara real-time

Jarak ke Burung

0.00
meter

Cara Penggunaan:

  1. Klik tombol "Aktifkan Kamera" dan izinkan akses ke kamera
  2. Masukkan perkiraan ukuran burung (dalam cm) - default 30cm
  3. Klik "Kalibrasi" untuk menyimpan setting
  4. Arahkan kamera ke burung yang ingin diukur jaraknya
  5. Klik pada burung di layar untuk mengukur jarak
  6. Gunakan tombol "Ambil Gambar" untuk menyimpan hasil

Tips: Untuk hasil terbaik, pastikan Anda mengetahui perkiraan ukuran burung target. Semakin akurat perkiraan ukuran, semakin akurat pula perhitungan jaraknya.
