Cara Penggunaan:

  1. Klik tombol "Aktifkan Kamera" dan izinkan akses ke kamera
  2. Pilih mode pengukuran (Burung atau Umum)
  3. Arahkan kamera ke target dan tunggu hingga stabil (indikator hijau)
  4. Klik pada target di layar untuk mengukur jarak
  5. Gunakan tombol "Ambil Gambar" untuk menyimpan hasil

Tips Akurasi: Pastikan kamera stabil sebelum melakukan pengukuran. Gunakan mode Burung untuk target kecil dan mode Umum untuk target besar. Hasil akan konsisten untuk objek yang sama pada kondisi pencahayaan yang sama.
